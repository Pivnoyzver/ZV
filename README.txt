Для работы программы необходимо изменить device файл с айпи адресами. Туда необходимо записать айпи адрес устройства работающего как приемник(не просто айпи адрес, а адрес в локальной сети).

Посмотреть его можно:
параметры > сеть и интернет > свойства > IPv4-адрес

Адреса сейчас в фале нужны исключительно для отладки и не имеют ничего общего с реальными адресами.


Для запуска режима приемника использовать следующую команду в терминале:

gst-launch-1.0 udpsrc port=5004 ! application/x-rtp, media=audio, clock-rate=44100, encoding-name=L16, channels=2 ! rtpL16depay ! queue ! audioconvert ! audioresample ! autoaudiosink
